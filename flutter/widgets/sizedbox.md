# SizedBox

Cria um widget invisível, adiciona um espeçamento em uma lista, também permite aumentar a altura do botão, colocando o botão como filho e definindo o height.

```dart
SizedBox(
  width: 64,
  height: 32,
  child: <Widget?>
)
```
