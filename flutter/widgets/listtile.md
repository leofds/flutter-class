# ListTile

Um item da lista, limita as informações exibidas

```dart
ListTile(
  title: Text('Tarefa 1'),
  subtitle: Text('20/11/2022'),
  leading: Icon(Icons.save, size: 30,),
  onTap: (){
    print('tarefa 1');
  },
)
```
