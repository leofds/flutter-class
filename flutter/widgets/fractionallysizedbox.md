# FractionallySizedBox

Um Widget que dimensiona seu filho para uma fração do total disponível

```dart
FractionallySizedBox(
  widthFactor: 0.5,
  heightFactor: 0.5,
  alignment: FractionalOffset.center,
  child: <Widget>
)
```
