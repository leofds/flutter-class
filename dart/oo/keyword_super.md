# Keyword super

Refere-se a classe Pai, pode ser um construtor, atributo ou m√©todo.

```dart

  String nome;
  double peso;

  Animal(this.nome, this.peso);
}

class Cachorro extends Animal {
  Cachorro(super.nome, super.peso);
```
