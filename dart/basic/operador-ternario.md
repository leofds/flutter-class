# Operador ternário

O operador ternário é um teste condicional assim com o if, porém efetuado em apenas uma linha, retornando os valores programados.

```dart
int media = 8;

String situacao = (media >= 7) ? 'Aprovado' : 'Reprovado';

print(situacao);    // Exibe: Aprovado
```
