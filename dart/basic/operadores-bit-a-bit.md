# Operadores bit a bit

| Operador | Descrição |
| -------- | --------- |
| & |E (em inglês AND) |
| \| | Ou (em inglês OR) |
| ^| Ou exclusivo (em inglês XOR) |
| ~ | Complemento (1 torna-se 0 e vice-versa) |
| << | Rotaciona à esquerda |
| >> | Rotaciona à direita |
| >>> | Rotaciona à direita sem sinal |
