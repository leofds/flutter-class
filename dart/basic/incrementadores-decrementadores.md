# Incrementadores/decrementadores

Forma reduzida de excrever a operação.

| Operador | Uso | Equivalente à |
| -------- | --- | ------------- |
| ++ | contador++ | contador = contador + 1 |
| -- | contador-- | contador = contador - 1 |

```dart
int contador = 10;
print(contador++);    // 10
print(contador);      // 11
print(++contador);    // 12
```
