# Função Print 

Escreve no terminal a mensagem "Olá mundo"

```dart
void main() {
  print('Olá mundo');
}
```

Utilize aspas duplas `"` ou simples `'`.

```dart
print('Com "aspas simples"');
print("Com 'aspas duplas'");
print('Com \'aspas simples\'');
```
