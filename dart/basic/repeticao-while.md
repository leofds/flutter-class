# Laço de repetição ENQUANTO (while)

Enquanto a condição entre os parênteses () for true, executa o bloco de código entre as chaves { }.

```dart
int contador = 0;
while (contador < 3) {
  print('Contador $contador');
  contador++;
}
// Resultado:
// Contador 0
// Contador 1
// Contador 2
```
