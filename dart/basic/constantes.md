# Constantes

Constantes armazenam um valor que não pode ser alterado durante a execução do programa.

**Const** valor definido em tempo de compilação

```dart
const nome = 'Mariana';
print(nome);
```

**Final** valor definido em tempo de execução

```dart
final int number;
number = 10;
print(number);
```
